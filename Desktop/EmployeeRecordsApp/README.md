## Project Title
Employee Records Management Desktop Application

## Description
A Java Swing application using JDBC to manage employee records. Supports CRUD operations with data validation.

## Features
- Add, update, delete employees
- View employees in JTable
- Input validation using RegEx
- Clean OOP structure

## How to Run
1. Import project into your IDE (Eclipse/IntelliJ)
2. Run `create_employee_table.sql` in MySQL
3. Update DBConnection.java if needed
4. Run `MainFrame.java`
